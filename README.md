Модуль 4: Финальный проект "Сеть для АгроТех"

Тема: Интеграция знаний по SDN и SD-WAN.

Задание 4: "Проектирование и защита" (Командное, 3 недели)

-   Цель: Разработать комплексное решение для гипотетической компании.
-   Задача: Объединить команды в 4 большие группы по 4 человека. Каждая группа получает один и тот же кейс, но должна предложить свое решение.

Кейс:
Компания "АгроТех" имеет: 1) Центральный офис/ЦОД (высокопроизводительные серверы), 2) Удаленный филиал на ферме (датчики, камеры видеонаблюдения), 3) 5 удаленных сотрудников. Задачи: обеспечить безопасный доступ к приложениям в ЦОДе для филиала и удаленных сотрудников, приоритезировать трафик видеоконференций, обеспечить отказоустойчивость каналов связи филиала (использует 4G и спутниковый канал).

 -  Что нужно сделать группе:
    1.  Спроектировать архитектуру: Нарисовать схему сети, выделить плоскости данных и управления.
    2.  Выбрать технологии: Обосновать, где будет использоваться SDN (например, в ЦОДе), а где SD-WAN (для подключения филиала и удаленных сотрудников). Предложить конкретные решения (например, ONOS для ЦОДа, решение на базе ZeroTier/Tailscale для удаленных сотрудников).
    3.  Смоделировать ключевые сценарии: Разработать демонстрацию на виртуальных стендах (Mininet, облачные VM) для одного-двух ключевых сценариев (например, автоматическое переключение канала в филиале при обрыве 4G).
    4.  Рассчитать бюджет: Оценить стоимость решения (можно использовать условные цифры, главное - методология).
-   Результат:
    -   Презентация проекта: 15-минутное выступление перед "инвесторами" (преподавателем и другими студентами) с защитой своего решения.
    -   Техническая документация: Краткое описание архитектуры и конфигураций.
    -   Демонстрационный стенд: Код и конфигурации для показа работоспособности ключевых функций.


Как распределить роли внутри больших групп
В каждой группе назначаются:
-  Team Lead: отвечает за общую архитектуру и презентацию.
-  Network Designer: разрабатывает схему сети, выбирает технологии.
-  Simulation Engineer: строит демо-стенд (Mininet, облачные VM).
-  Documentation & Budget Specialist: готовит техдокументацию и расчёт бюджета.
